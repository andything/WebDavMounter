# WebDAV Mounter v1.0 正式发布 🎉

WebDAV Mounter 是一款 macOS 状态栏工具，帮助您将 WebDAV 服务器轻松挂载为本地磁盘，像使用本地文件夹一样访问远程文件。

---

## ✨ 主要功能

- **一键挂载**：通过菜单栏快速挂载/卸载 WebDAV 连接
- **自动挂载**：支持开机自动挂载，可自定义启动延迟时间
- **多配置管理**：支持添加、编辑、删除多个 WebDAV 配置
- **状态可视化**：菜单栏和配置窗口实时显示挂载状态
- **安全存储**：凭据保存在 macOS 钥匙串中，安全可靠

---

## 🖥️ 系统要求

- macOS 12 Monterey 或更高版本
- Apple Silicon 或 Intel 芯片

---

## 📥 安装方法

1. 下载 `WebDavMounter.dmg` 文件
2. 打开 `.dmg` 文件，将 `WebDavMounter.app` 拖入 `Applications` 文件夹
3. 首次打开时，如提示"无法验证开发者"，请在系统设置中允许运行
4. 点击菜单栏图标开始配置

---

## 📝 使用说明

1. 点击菜单栏图标 → **配置设置**
2. 点击 **添加**，填写 WebDAV 信息：
   - 显示名称（如：我的云盘）
   - 域名（如：webdav.example.com）
   - 端口（如：15006）
   - 用户名 / 密码
3. 保存后，在菜单栏选择 **手动挂载** 即可
4. 如需开机自动挂载，在配置设置中勾选 **启动时自动挂载**
