## ✨ 功能特征

**🔗 WebDAV 挂载**  
将远程 WebDAV 服务器挂载为 macOS 本地磁盘，通过 Finder 直接访问、读写远程文件，像使用本地文件夹一样简单。支持标准 WebDAV 协议，兼容主流 WebDAV 服务器。支持添加多个 WebDAV 配置，独立管理，轻松切换。

**🚀 自动挂载**  
支持开机自动挂载所有配置，可自定义启动延迟（1-30 秒），确保网络就绪后再进行挂载，无需手动操作。

**🔒 安全存储**  
用户名和密码均保存在 macOS 钥匙串中，采用系统级加密，安全可靠。

**📊 状态可视化**  
配置窗口和菜单栏实时显示每个连接的挂载状态，"已连接 ✓"绿色标签一目了然。

**🎨 原生体验**  
完美适配 macOS 深色/浅色模式，圆角窗口设计，轻量级，资源占用极低。

**💻 系统兼容**  
支持 macOS 12 Monterey 及以上，兼容 Apple Silicon 和 Intel 双平台。

## 📥 安装方法

1. 下载 `WebDavMounter.dmg` 文件  
2. 打开 `.dmg`，将 `WebDavMounter.app` 拖入「应用程序」文件夹  
3. 首次打开如提示"无法验证开发者"，请在系统设置中允许运行  
4. 点击菜单栏图标开始配置

---

## 📝 使用说明

1. 点击菜单栏图标 → **配置设置**  
2. 点击 **添加**，填写 WebDAV 信息  
3. 保存后，在菜单栏选择 **手动挂载** 即可  
4. 如需开机自动挂载，勾选 **启动时自动挂载**

---

## 🖱️ 菜单栏快捷管理

应用常驻 macOS 菜单栏，点击图标即可快速挂载、断开连接、打开配置或退出应用，所有操作触手可及。
